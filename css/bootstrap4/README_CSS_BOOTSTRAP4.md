## Сетка

### Ключевые классы

* `container` - класс контейнера с ограниченной максимальной шириной
* `container-fluid` - класс контейнера с максимальной шириной 100%
* `row` - класс ряда
* `col` - класс ячейки

### Breakpoints

* `(.col-)` **Extra small** - **< 576px**
* `(.col-sm-)` **Small** - **>= 576px**
* `(.col-md-)` **Medium** - **>= 768px**
* `(.col-lg-)` **Large** - **>= 992px**
* `(.col-xl-)` **Extra Large** - **>= 1200**
